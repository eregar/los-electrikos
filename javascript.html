JavaScript



JavaScript es un lenguaje de programación que permite a los desarrolladores crear acciones en sus páginas web. Es utilizado mayormente para ejecutar animaciones y funcionamientos de botones u otras características de una página, es decir, sirve para darle a una página web mas interacción con el usuario y la posibilidad de tener mas servicios que ofrecer, permitiendo así un desarrollo con mayor versatilidad y mas facilidad a la vista.

In Document.

El objeto document es el que tiene el contenido de toda la página que se está visualizando. Esto incluye el texto, imágenes, enlaces, formularios, ... Gracias a este objeto vamos a poder añadir dinámicamente contenido a la página, o hacer cambios, según nos convenga.

CDN.

CDN (Content Delivery Network) es el que es subido a un grupo de servidores distribuidos en lugares distintos. estos servidores almacenan duplicados de información para poder así satisfacer solicitudes de esta, en cualquier servidor que este mas proximo al usuario. Proveen un servicio rápido y evaden la saturación.

As Library

Este es un método que se basa en tener el código dentro de una librería de Java para trabajar en base a información que es importada.
